Module: BackToTopButton

This React component provides a "Back to Top" button, which appears when the user scrolls down the page. Clicking the button smoothly scrolls the page back to the top. This component is useful for improving user navigation on long pages. It uses conditional rendering based on the scroll position to either show or hide the button.
