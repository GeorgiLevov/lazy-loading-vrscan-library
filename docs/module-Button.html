Module: Button

This component renders a customizable button that adapts its appearance based on provided props. It supports multiple variants, sizes, and the ability to include an icon. The button can be rendered as a standard button or as a link (using react-router's Link component) based on the provided props.
Properties:
Name Type Attributes Default Description
id string <optional>
 Optional ID for the button.
variant string The variant of the button, which defines its styling.
size string <optional>
 'medium' The size of the button, influencing padding and font size.
icon object <optional>
 Icon component to be rendered inside the button.
iconfirst boolean <optional>
 Determines if the icon is positioned before the text.
href string <optional>
 URL for the button link. If provided, the button is rendered as a Link.
children ReactNode The content inside the button.
onClick function <optional>
 The function to be executed on button click.
selected boolean <optional>
 false Indicates if the button is selected (for toggleable buttons).
style object <optional>
 Custom style object to override default styles.
