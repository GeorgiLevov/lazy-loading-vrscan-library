Module: GuestRoutes

This React component handles routing for guests (unauthenticated users) in the application. It uses the user's authentication status from Redux state to determine which routes should be accessible to a guest. If a user is authenticated, they are redirected to the home page; otherwise, they can access guest routes.

(static) renderRoutes() → {JSX.Element}

Determines the component to render based on the authentication status of the user. If the user's status is 'success' and they are authenticated, the function redirects to the home page. If the user's status is not 'success' or they are not authenticated, it renders the child routes (Outlet).
