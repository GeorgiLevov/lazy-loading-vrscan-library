Module: Card

This React component renders a card that can display various types of content based on the provided variant. It supports multiple styles like base, inverted, profile, and vrscan. The component is highly customizable, allowing for different layouts and content such as images, summaries, and additional children elements.
Properties:
Name Type Attributes Description
scanId number <optional>
 Optional ID for the card, used in VRScan cards.
summary array <optional>
 Array containing summary details for the card.
name string <optional>
 Name to be displayed on the card.
variant string Variant of the card to determine its style.
imageAlt string <optional>
 Alt text for the image.
imageSrc string | array <optional>
 Source for the image or array of images.
favorited bool <optional>
 Indicates if the item is favorited (for vrscan cards).
toggleFavorite func <optional>
 Function to toggle the favorite status (for vrscan cards).
children ReactNode <optional>
 Additional children elements to be rendered inside the card.
